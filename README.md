# تطبيق كنوز مصر للهدايا - Flutter

تطبيق أندرويد احترافي لمتجر كنوز مصر للهدايا الفاخرة - أطقم القهوة والإستكانات والفناجين المصرية الأصيلة.

## الميزات الرئيسية

- 🎁 **جمع الهدية بنفسك**: اختر البوكس والهدايا والتغليف حسب رغبتك
- 📦 **البوكسات الجاهزة**: أقسام متعددة حسب المناسبات (الزفاف، عيد الأم، التخرج، إلخ)
- ☕ **أطقم القهوة المميزة**: تشكيلة واسعة من أطقم القهوة الفاخرة
- 🎉 **العروض الحصرية**: خصومات وعروض خاصة على المنتجات المختارة
- 🛒 **العربة الذكية**: إدارة سهلة للمنتجات والكميات
- 👤 **حسابي الشخصي**: متابعة الطلبات والحالات

## المتطلبات

- Flutter 3.0 أو أحدث
- Dart 3.0 أو أحدث
- Android SDK 21 أو أحدث

## البناء والتشغيل

### تثبيت الاعتماديات

```bash
flutter pub get
```

### تشغيل التطبيق على جهاز أو محاكي

```bash
flutter run
```

### بناء APK للإصدار

```bash
flutter build apk --release
```

سيتم حفظ APK في: `build/app/outputs/flutter-apk/app-release.apk`

## البناء عبر Codemagic

يمكن بناء APK تلقائياً عبر Codemagic:

1. اذهب إلى [Codemagic.io](https://codemagic.io)
2. سجل الدخول بحسابك على GitHub
3. اختر هذا المستودع
4. سيتم بناء APK تلقائياً عند كل push

## هيكل المشروع

```
lib/
├── core/
│   ├── config/       # إعدادات التطبيق
│   └── theme/        # المظهر والألوان
├── data/
│   ├── models/       # نماذج البيانات
│   ├── services/     # خدمات الاتصال
│   ├── controllers/  # المتحكمات
│   └── repositories/ # المستودعات
└── presentation/
    ├── screens/      # الشاشات
    └── widgets/      # المكونات المشتركة
```
